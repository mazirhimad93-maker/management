# **App Name**: CreatorFlow

## Core Features:

- Hero Section with Looping Video: Display a full-width hero section with a muted looping background video and dark overlay.
- Video Popup Modal: Open a video in a modal triggered by the 'Watch the Video' buttons, clearing the iframe source on close to stop playback.
- Native Booking Flow Modal: Implement a 2-step booking modal (date/time selection + form questions) to schedule valuation calls.
- Automated Timezone Detection: Auto-detect the visitor's timezone for the booking scheduler and allow manual override.
- Webhook Integration: Submit booking form data via POST request to a configured webhook URL to create Google Calendar events.
- Client Case Studies: Showcase client success stories in a visually balanced card layout.

## Style Guidelines:

- Background: Dark zinc/near-black for a dark mode theme (#18181b).
- Text: White/zinc shades for readability against the dark background (#f4f4f5).
- Accent: A deep purple (#a855ba) will be used for CTAs and interactive elements, providing contrast while remaining harmonious with the dark theme.
- Body text font: 'Inter' sans-serif for clear and modern readability.
- Headline font: 'Space Grotesk' sans-serif for a techy feel. To be paired with Inter if longer text is needed.
- Use social-style verified badge with small check icon for trust indicators.
- Components: Cards with rounded-xl / rounded-2xl for a modern, soft feel.
- Borders: Subtle zinc outlines to define sections without harsh lines.
